Detailed Description

template<class T>
class ibmppl::linear_task_partitioner< T >

implement block-based linear partitioning for container class T

class <T> implement the following interface: T::size(), T::begin(), T::end(), T::iterator, and T::const_iterator

ptype:

  • STATIC_PARTITION: T is divided into <num_threads> # of tasks equally
  • BLOCK_PARTITION: each task is of size <n> elements

chunk_size:

  • for STATIC_PARTITION: chunk_size is minimal chunk size for task (if not enough tasks, # of tasks will be less than # threads
  • for BLOCK_PARTITION: chunk_size is the task chunk size
